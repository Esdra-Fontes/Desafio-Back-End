🌐 Desafio Back-End – Node.js
---
Este repositório contém um projeto back-end desenvolvido com Node.js, criado como um desafio prático para reforçar conhecimentos em desenvolvimento de APIs, arquitetura de servidores e integração de sistemas.

📌 Objetivo do Projeto:
---
O objetivo deste desafio é demonstrar a estrutura e funcionamento de uma aplicação back-end, incluindo organização de rotas, controllers, gerenciamento de requisições HTTP e configuração de um servidor moderno utilizando JavaScript.

🛠️ Tecnologias Utilizadas:
---
- Node.js – Ambiente de execução JavaScript no servidor
- Express.js – Framework para criação de APIs
- JavaScript (ES6+) – Lógica da aplicação
- Nodemon – Ferramenta para desenvolvimento com reload automático
- npm – Gerenciamento de dependências

📁 Estrutura do Projeto:
---
Desafio-Back-End/
│
├── controllers/ – Lógica das requisições
├── routes/ – Definição das rotas da API
├── models/ – Estrutura de dados (se aplicável)
├── config/ – Configurações da aplicação
├── .gitignore
├── package.json
├── server.js (ou app.js) – Arquivo principal do servidor
└── README.md

▶️ Como Executar o Projeto:
---
- Clone este repositório: git clone https://github.com/Esdra-Fontes/Desafio-Back-End.git
- Acesse a pasta do projeto: cd Desafio-Back-End
- Instale as dependências: npm install
- Inicie o servidor: npm run dev (ou npm start, dependendo da configuração)
- Abra no navegador: http://localhost:3000

📚 Aprendizados:
---
- Criação de APIs REST com Node.js e Express
- Organização de código em camadas (rotas, controllers, etc.)
- Manipulação de requisições e respostas HTTP
- Uso de ferramentas modernas de desenvolvimento back-end
- Estruturação de projetos escaláveis
